# 技术设计

## 技术选型

- 语言：Python 3.13
- 桌面界面：Tkinter
- 图片处理：Pillow
- 存储方式：JSON 索引 + 图片文件

选择理由：当前电脑已经具备 Python、Tkinter 和 Pillow，可先不依赖联网安装新库，适合小步开发和后续打包。

## 数据存储

用户指定的保存文件夹内包含：

- `history_index.json`：历史记录索引。
- `images/`：保存原图。
- `thumbs/`：保存缩略图。

程序目录内包含：

- `clipboard_history_config.json`：保存用户选择的存储目录、保存天数、排序方式、筛选方式和上次清理提醒日期。

## 历史记录字段

- `id`：唯一编号。
- `type`：`text` 或 `image`。
- `created_at`：首次记录时间。
- `last_copied_at`：最近一次复制时间。
- `pinned`：是否置顶。
- `note`：备注。
- `content_hash`：用于识别重复内容。
- `text`：文字内容。
- `image_path`：原图路径。
- `thumb_path`：缩略图路径。
- `source_name`：图片来源文件名，可为空。

## 剪贴板策略

- 使用定时轮询读取剪贴板。
- 优先读取图片，再读取文字。
- 重复内容不新增卡片，只更新时间。
- 复制回剪贴板时，文字使用 Tkinter 剪贴板接口；图片在 Windows 上写入 DIB 格式。

## 清理策略

- 只清理未置顶记录。
- 以 `last_copied_at` 判断是否过期。
- 启动时或每天首次运行时检查过期记录。
- 删除前弹窗提醒，用户确认后再删除。
