# 项目指引

## 项目目标

开发一个简洁、直观、便携的 Windows 剪贴板历史工具，帮助用户找回近期复制过的文字和图片。

- 技术设计：`docs/02_技术设计.md`

- 工作说明：`docs/04_工作说明.md`
- 打包与分发说明：`docs/05_打包与分发说明.md`
- UI 与 OCR 优化说明：`docs/06_UI与OCR优化说明.md`

- 每日开发日志：`开发日志/YYYY-MM-DD.md`

## 工作原则

- 小步开发，每一步完成后尽量能运行或检查。
- 优先保证稳定、安全、数据不丢失。
- OCR 保持为可选本地组件，不做联网识别、自动粘贴和安装器。
- 开机自启保持用户可选，默认关闭。
- 涉及用户数据的操作必须谨慎，自动清理前要提醒确认。
