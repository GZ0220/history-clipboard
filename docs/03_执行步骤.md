# 执行步骤

## 第 1 步：项目基础

- 创建项目目录、源码目录、文档目录和开发日志目录。
- 建立需求说明、技术设计和执行步骤文档。
- 建立每日开发日志。

## 第 2 步：核心存储

- 实现配置读取和保存。
- 实现历史索引读取和保存。
- 实现文字、图片、缩略图保存。
- 实现重复内容合并。

## 第 3 步：基础界面

- 实现主窗口。
- 实现顶部搜索、排序、类型筛选和保存天数控件。
- 实现历史卡片列表。

## 第 4 步：核心交互

- 实现复制回剪贴板。
- 实现置顶、删除和图片备注。
- 实现过期清理提醒。

## 第 5 步：验证与整理

- 编写基础自动化测试。
- 运行语法检查和测试。
- 更新 README 和开发日志。
- 确认下一阶段待办。

## 第 6 步：打包为便携 exe

- 安装 PyInstaller。
- 使用 `历史粘贴板.spec` 生成 `dist/历史粘贴板/`。
- 在本机双击 `历史粘贴板.exe` 验证。
- 将整个 `dist/历史粘贴板/` 文件夹复制到其他 Windows 电脑测试。
