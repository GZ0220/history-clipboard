# 需求说明

## 核心需求

- 软件运行在 Windows 电脑上，作为便携工具使用。
- 双击运行后开始记录剪贴板，关闭软件后停止记录。
- 首次启动时让用户选择历史数据保存文件夹，后续自动记住。
- 记录最近复制的文字和图片。
- 支持设置保存天数，例如 1 天、3 天、5 天、7 天、30 天和自定义天数。
- 超过保存天数的未置顶记录，需要先提醒用户确认，再删除。
- 置顶记录永久保留，不参与自动清理。

## 历史查看

- 历史内容以卡片形式展示。
- 支持按时间升序或降序排列。
- 支持按类型查看全部、文字、图片。
- 支持搜索文字内容。
- 图片卡片支持添加备注，并支持搜索备注。

## 卡片操作

- 复制：把历史内容重新放回系统剪贴板，由用户在目标软件中手动粘贴。
- 置顶：保留重要记录，避免被清理。
- 删除：删除单条历史记录和关联图片文件。
- 备注：图片记录可添加或修改备注。

## UI 要求

- 中文界面。
- 主要颜色为淡蓝色和白色。
- 简洁直观，避免复杂入口。
